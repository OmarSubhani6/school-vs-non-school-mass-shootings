# School-Related vs Non-School-Related Mass Shootings in the United States

## Overview
This project analyzes differences between school-related and non-school-related mass shooting incidents in the United States using publicly available data from Stanford University.

## Data
- Source: Stanford University Mass Shootings Dataset
- File: shootings.csv
- Data includes incident location, year, and contextual characteristics

## Tools & Technologies
- R
- RStudio
- ggplot2
- Data cleaning and exploratory data analysis (EDA)

## Analysis
- Cleaned and transformed raw shooting data for analysis
- Compared incident trends by location type
- Visualized temporal and categorical differences using ggplot2
- Identified patterns in distribution and frequency across contexts

## Key Skills Demonstrated
- Data wrangling and cleaning
- Exploratory data analysis
- Information visualization
- Translating data into insights
